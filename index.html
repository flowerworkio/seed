Welcome to Flowerwork!

At Flowerwork we believe the best software is made by teams not individuals. Flowerwork helps software teams work together and ship everyday. Our platform empowers software teams to shorten their feedback loops, cross-pollinate their best ideas, and ship everyday. We create ready to deploy container images for every push to GitHub and collaborative, onlinepreview environments for every GitHub pull request.

If you are seeing this image it is because we are setting up a container registry or a new deployment. This allows us to deploy more quickly when the final code is ready to deploy. We appreciate your patience why you wait for your code to be live.
